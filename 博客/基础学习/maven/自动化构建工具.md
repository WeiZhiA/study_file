# 基础知识

DSL 是域特定语言的首字母缩写词，帮助我们编写声明性代码来减少样板文件。用 DSL 编写的代码会更容易阅读。

DSL语言的常见用法是Web开发中的HTML，构建工具中的Gradle，数据管理中的SQL ，标记语言的XML等。


## Kotlin DSL

Kotlin DSL 建立在核心语言 Kotlin 之上。


好处：

良好的可读性
轻松适应 Kotlin 语法
代码导航和自动建议
编译时错误
目前支持依赖和工件配置、源集等的类型安全模型访问器
